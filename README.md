# Grab the Treasure

**Grab the Treasure** is a 2D pixel-art auto-runner platformer built with Unity.
The character runs automatically while the player controls jumps with **space** or **mouse click**.  
Avoid obstacles, collect coins, and reach the treasure chest to complete each level.

## Controls
- **Space** or **Left Mouse Click** — Jump  
- Click on the screen to select a level.

## Unity Version
- **v2019.1.3f1**
